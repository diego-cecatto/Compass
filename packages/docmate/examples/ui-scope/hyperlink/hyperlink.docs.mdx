A component that does something special and renders text in a div.

### Component usage

```js
import { HyperLink } from './src/hyperlink';
<HyperLink
    link={'https://www.google.com.br'}
    openInNewTab={true}
    children={<>Click here</>}
    showTooltip={false}
/>;
```
